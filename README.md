# REACT.JS + TYPESCRIPT - Building frontend for web Trello-clone-app-frontend

## Requirements

**nodejs >= v14.7.0
**npm >= v6.14.7
\*\*yarn > v1.19.1

## Clone project and run test

```
$ git clone https://github.com/HopNgo/trello-clone-app-frontend.git
$ cd folder
$ npm install ... (packages in dependencies)
$ npm start

```
